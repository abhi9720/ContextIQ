# Product Wiki

This document provides a high-level overview of the product, its features, and how to use it.

## Overview

The application is a web-based tool that allows users to upload documents and generate quizzes and flashcards from them. The goal is to provide a simple and efficient way to create learning materials from existing documents.

## Features

- **Document Upload:** Users can upload documents in various formats (e.g., PDF, DOCX, TXT).
- **Quiz Generation:** Users can generate quizzes from their uploaded documents, with options to specify the difficulty, number of questions, and question types.
- **Flashcard Generation:** Users can generate flashcards from their uploaded documents, with options to specify the number of flashcards.

## How to Use

1. **Upload a document:** Go to the "Documents" page and upload a new document.
2. **Wait for processing:** The document will be processed in the background. You can check the status of the document on the "Documents" page.
3. **Generate content:** Once the document is processed, you can generate a quiz or flashcards from it by clicking the corresponding buttons.
4. **View content:** The generated content will be available on the "Quizzes" or "Flashcards" page.
